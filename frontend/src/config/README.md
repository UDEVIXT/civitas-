## /config
Esta carpeta contiene archivos de configuración global y variables de entorno que definen el comportamiento de la aplicación en diferentes entornos (dev, staging, prod).

### Ejemplos 
Configuración de Axios, constantes globales, temas de diseño o configuración de Firebase/Supabase.

### Regla 
No debe contener lógica de negocio, solo definiciones y exportación de instancias configuradas.
